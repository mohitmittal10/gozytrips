# **App Name**: Odyssey Luxe

## Core Features:

- Floating Navigation Bar: Implement a glass-style navbar that appears on scroll, including links to 'Home', 'About Us', 'Testimonials', 'Contact Us', 'Packages', and 'Gallery'.
- Destination Packages Display: Showcase luxurious travel packages with high-definition images and detailed descriptions.
- Image Gallery: Implement a gallery to display travel destinations. The gallery can accept various images from different destinations to allow users to view them.
- Google Reviews Integration: Integrate Google Reviews on the homepage to display customer testimonials and ratings.
- Relevant Footer Links: Add a footer with links to essential pages like contact information, privacy policy, terms of service, etc.
- AI Architect: Create a simple input form on the page where users enter: Destination, Number of Days, and 'Vibe' (e.g., Chill, Adventure, Foodie). When 'Generate' is clicked, it should simulate an AI processing state with a 'shimmer' glass animation. The result must be displayed on the same page (not a new tab) using a clean, vertical timeline component. Include a 'Book Taxi for this Day' button inside each itinerary card to cross-sell the taxi service.

## Style Guidelines:

- Primary color: Midnight Navy (#191970) for a luxurious feel.
- Background color: Desaturated Midnight Navy (#2E2E8B) to complement the primary color.
- Accent color: Soft Sand-Gold (#F0E68C) as an analogous accent to evoke sophistication and elegance.
- Font pairing: 'Playfair' (serif) for headlines and short text chunks, combined with 'PT Sans' (sans-serif) for body text. This combination supports an elegant feel alongside readable body text.
- Use Glassmorphism: Implement semi-transparent 'frosted glass' cards with a soft background blur.
- Incorporate thin-line minimalist icons to maintain a clean, ultra-modern aesthetic.
- Add subtle animations to create a sophisticated and immersive user experience.
